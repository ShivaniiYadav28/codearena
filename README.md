# 🚀 CodeArena

CodeArena is a full-stack coding platform inspired by LeetCode.

## Features

✅ User Authentication (JWT)

✅ Online Code Editor

✅ Run Code

✅ Submit Solutions

✅ Dashboard

✅ Leaderboard

✅ Track Submissions

## Tech Stack

### Frontend
- React.js
- React Router
- Axios

### Backend
- Node.js
- Express.js
- MongoDB
- JWT Authentication
